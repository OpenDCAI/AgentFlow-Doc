常见问题

Q: 什么时候用 APITool？

当你的工具：

  • 调用外部 HTTP API
  • 不需要保持状态
  • 不需要预加载资源
  • 每次调用相互独立

Q: 什么时候用 Backend？

当你的工具：

  • 需要预加载模型或资源
  • 需要维护长连接
  • 需要为每个用户分配独立资源
  • 需要管理资源生命周期

Q: 什么时候实现 warmup()？

当你需要：

  • 加载 ML 模型
  • 建立连接池
  • 初始化全局缓存
  • 启动后台进程

Q: 什么时候实现 initialize()？

当你需要：

  • 为每个用户分配独立资源
  • 维护用户级别的状态
  • 隔离用户之间的操作
